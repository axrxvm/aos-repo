# i386 Architecture

This directory contains packages, modules, and files for the i386 (32-bit x86) architecture.

## Structure

- **iso**: ISO images for i386 architecture
- **akmcc**: AOS Kernel Module Compiler Collection packages
- **kmodule**: Kernel modules for i386
- **pkgs**: Software packages compiled for i386
- **etc**: Configuration files and miscellaneous items
- **tmp**: Temporary files and builds

Each category folder contains items specific to the i386 architecture.
