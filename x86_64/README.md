# x86_64 Architecture

This directory contains packages, modules, and files for the x86_64 (64-bit x86) architecture.

## Structure

- **iso**: ISO images for x86_64 architecture
- **akmcc**: AOS Kernel Module Compiler Collection packages
- **kmodule**: Kernel modules for x86_64
- **pkgs**: Software packages compiled for x86_64
- **etc**: Configuration files and miscellaneous items
- **tmp**: Temporary files and builds

Each category folder contains items specific to the x86_64 architecture.
